grove

Open tools for structured research. Use with wheat, or use standalone.

$ npm install @grainulator/grove
Tools

detect-sprints

Find sprint directories by scanning for claims.json files. Uses git history to rank and pick the active sprint.

grove detect-sprints --json grove detect-sprints --active grove detect-sprints --root /repo

generate-manifest

Build a wheat-manifest.json that maps topics to claims and files. One file, entire sprint state.

grove generate-manifest grove generate-manifest --root /repo grove generate-manifest --out m.json

build-pdf

Convert markdown to PDF via npx md-to-pdf. No local install needed.

grove build-pdf output/brief.md
Templates

explainer.html

Dark scroll-snap presentation. Each section fills the viewport, snaps on scroll. Cards, columns, tables built in.

scroll-snap dark theme responsive self-contained dot navigation

dashboard.html

Sprint status dashboard with phase progress, evidence bars, conflict tracking, and claim inventory table.

grid layout evidence bars typed badges responsive compilation bar
Principles

Zero dependencies

Node built-in modules only. No npm install surprises, no supply chain concerns.

Git as truth

Sprint detection reads git log, not config files. If git knows it, we do not duplicate it.

Self-describing

New sessions understand the repo without full scans. The manifest is the entry point.

AI-friendly

Works with Claude Code search tools (Glob, Grep, Read) out of the box. Manifest under 10KB.
